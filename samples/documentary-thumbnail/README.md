# Documentary Thumbnail Concept

![Who Owns the Moon thumbnail](who-owns-the-moon.png)

This 1920 x 1080 self-initiated concept demonstrates a documentary/educational thumbnail direction using manual layout, typography, masking, color treatment, and public-domain NASA photography. It is not client work and makes no CTR or performance claim.

The fictional premise, "Who Owns the Moon?", is structured around politics, business, and power. A real client version would be adapted to the supplied script, title, channel identity, approved photo sources, and A/B testing plan.
