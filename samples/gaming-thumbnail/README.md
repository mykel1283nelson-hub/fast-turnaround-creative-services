# Gaming Thumbnail Concept

![Level 1 versus Final Boss thumbnail](level-1-vs-final-boss.png)

This 1280 x 720 self-initiated concept uses a fictional player avatar and boss character. It demonstrates mobile-readable hierarchy, expressive faces, strong subject separation, and short high-contrast copy without presenting fictional CTR results or client work.

The layout can be adapted to supplied creator photos, game captures, channel colors, and the actual video premise. No copyrighted game art or fabricated analytics are used in this sample.
